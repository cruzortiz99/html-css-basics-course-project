Formularios

Los formularios son de suma importancia para las páginas o aplicaciones web. Se pueden usar para el registro de usuario, entrar en comunicación con el negocio, hacer algún pedido, etc.

Los formularios se generan en el HTML, sin embargo, los datos enviados deben ser manejados por JS y por alguna aplicación en el servidor

      form action="#" method="post" style="display: flex; flex-direction: column"
        label for="name"Nombrelabel
        input type="text" name="name" required autofocus/
        label for="mail"Correolabel
        input type="email" name="mail" required/
        label for="birthDate"Fecha de nacimientolabel
        input type="date" name="birthDate"/
        label for="ci"Cédulalabel
        input name="ci" type="number" min="5000000" max="40000000" required/

        textarea name="remark" cols="30" rows="30"textarea
