Imágenes

La imágenes forman parte importante de las páginas web. Lo importante de las imágenes en una página o aplicación web, es mantener las propociones o garantizar que las secciones importantes de la imagen se vean en el rango de tamaños requeridos

Para ello es importante, manejar las dimensiones y diferentes imágenes que puedan ser mostradas dependiendo del tamaño de la pantalla (esto se logran con CSS o JS para mejor rendimiento)
