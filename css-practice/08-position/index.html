Position

Permite definir la posición de los elementos del HTML pero con respecto al flujo del documento, no necesariamente al layout

  1. Static: sigue el flujo natural del documento
  2. Relative: es relativo al flujo natural del documento
  3. Absolute: relativo al elemento más cercano posicionado diferente al static que lo contiene
  4. Fixed
  5. Sticky
        .relative {
          position: relative;
          top: 15px;
        }
        .absolute {
          position: absolute;
          border-color: blue;
          border-style: solid;
          border-width: 5px;
          right: 0;
          bottom: 0;
        }
        .fixed {
          position: fixed;
          top: 20vh;
        }
        .sticky {
          position: sticky;
          top: 50vh;
        }
