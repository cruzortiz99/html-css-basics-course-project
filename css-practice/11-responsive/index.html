Responsive

Hacer una página responsive, refiere ha hacer que la página adapte su estructura o contenido acorde con el dispositivo que lo despliega

Esto se garantiza que el contenido relevante y asi como las funcionalidades se vean y sientan cómodas en cualquier dispositivo.

        body {
          display: grid;
          min-height: 100vh;
          grid-template-areas:
            'header'
            'nav'
            'main'
            'footer';
          grid-template-rows: 15vh 40px auto 40px;
        }
        .header {
          display: flex;
          justify-content: center;
          align-items: center;
        }
        .nav {
          position: sticky;
          top: 0;
        }
        .nav > ul {
          display: flex;
          flex-direction: row;
          justify-content: flex-start;
        }
        .nav > ul li a {
          display: flex;
          justify-content: center;
          align-items: center;
        }
        .nav > ul li a:hover {
          background-color: lightcyan;
          font-weight: bold;
        }
        .header {
          grid-area: header;
        }
        .nav {
          grid-area: nav;
        }
        .main {
          grid-area: main;
        }
        .footer {
          grid-area: footer;
        }
        @media screen and (min-width: 680px) {
          body {
            grid-template-areas:
              'header header'
              'nav main'
              'footer footer';
            grid-template-rows: 15vh auto 40px;
            grid-template-columns: 200px auto;
          }
          div {
            background-color: seagreen;
            border-color: darkblue;
          }
          .nav > ul {
            flex-direction: column;
          }
          .nav > ul li {
            height: auto;
            width: auto;
            margin: 5px 0 5px 0;
          }
        }
