CSS GRID

Las CSS grid son una forma de especificar el layout de la página. Permite el posicionamiento en 2D de los elementos de una página

De las virtudes más resaltantes, además de el posicionamiento 2D, es lo explicito del posiciones o el layout, lo flexible de la maya (GRID) y la capacidad de manejar el espácio en blanco

        .container {
          display: grid;
          grid-template-columns: 1fr 1fr 1fr;
          grid-template-rows: 1fr 1fr 1fr;
          grid-gap: 10px;
        }
        .container > div {
          background-color: lightblue;
          width: 100%;
          height: 100%;
        }
        .container > div:nth-child(1) {
          grid-column: 1 / 2;
          grid-row: 1 / 2;
        }
        .container > div:nth-child(2) {
          grid-column: 2 / 3;
          grid-row: 1 / 2;
        }
        .container > div:nth-child(3) {
          grid-column: 3 / 4;
          grid-row: 1 / 2;
        }
        .container > div:nth-child(4) {
          grid-column: 1 / 2;
          grid-row: 2 / 3;
        }
        .container > div:nth-child(5) {
          grid-column: 3 / 4;
          grid-row: 2 / 3;
        }
        .container > div:nth-child(6) {
          grid-column: 1 / 2;
          grid-row: 3 / 4;
        }
        .container > div:nth-child(7) {
          grid-column: 2 / 3;
          grid-row: 3 / 4;
        }
        .container > div:nth-child(8) {
          grid-column: 3 / 4;
          grid-row: 3 / 4;
        }
