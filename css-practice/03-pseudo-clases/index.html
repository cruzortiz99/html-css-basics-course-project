Pseudo-clases

Las pseudo-clases refieren a clases que se aplican a los elementos del html, pero que no son explícitamente declaradas en el mismo. Son muy útiles para definir un estilo, cuando ocurre alguna acción del usuario por ejemplo, pasar por encima de un div

      div  {
        background-color: lightblue;
        height: 100px;
        width: 100px;
      }
      div:hover {
        width: 120px;
        height: 120px;
      }
      div:first-child  {
        background-color: green;
      }
      div:last-child {
        background-color: green;
      }
