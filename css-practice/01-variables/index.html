CSS Variables

Las variables en CSS, son una de las características que permiten escalar las aplicaciones web. Permiten crear referencias de un valor, lo que permite que al cambiar el valor de la referencia, automáticamente se actualicen las demás características

      :root {
        --div-color: lightblue;
        --box-size-width: 30vw;
        --box-size-height: 30vh;
      }
      #app {
        height: var(--box-size-height);
        width: var(--box-size-width);
        background-color: var(--div-color);
      }
