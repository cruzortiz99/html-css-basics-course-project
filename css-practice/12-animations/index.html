Animaciones

Las animaciones y transiciones, son muy efectivas y mayor mente usadas para los momentos de espera del usuario.

Se ha confirmado que un segundo de espera en una aplicación causa el mismo efecto que una película de terror en el usuario, y según un los datos de Amazon, por cada segundo de espera en la respuesta de una app, llegan a perderse millones de dolares, porque el cliente busca otras opciones

Agregar animaciones, transiciones o algún elemento que le indique al usuario que su transacción esta en progreso es una de las estrategias que ha reducido los efectos nocivos de la espera en un aplicación o página web

Actualmente, se ha buscado agregar la interacción del usuario a cada elemento que se utilice para distraer la carga de datos, visuales, gráficas, etc.

Quizás los tiempos de respuesta no se puedan mejorar, pero se puede distraer al usuario para que note menos la espera.

        .container > div {
          background-color: lightblue;
          border-width: 2px;
          border-style: solid;
          width: 100px;
          height: 50px;
          position: relative;
          animation: moving;
          animation-duration: 5s;
          animation-iteration-count: infinite;
          animation-timing-function: ease-in-out;
          animation-fill-mode: backwards;
          transition: width 1s, height 1s;
        }
        .container > div:hover {
          width: 0px;
          height: 0px;
        }
        @keyframes moving {
          0% {
            top: -40px;
            left: -300px;
          }
          25% {
            top: -40px;
            left: 300px;
            background-color: darkred;
            transform: rotate(180deg);
          }
          50% {
            top: 40px;
            left: 300px;
            background-color: darkgreen;
            transform: rotate(180deg);
          }
          75% {
            top: 40px;
            left: -300px;
            background-color: darkblue;
            transform: rotate(0deg);
          }
          100% {
            top: -40px;
            left: -300px;
            background-color: lightblue;
            /* transform: rotate(-90deg); */
          }
        }
