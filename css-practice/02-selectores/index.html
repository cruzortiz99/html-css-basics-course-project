Selectores

Los selectores son fundamentales para CSS, es la manera que le permite distinguir cuando o sobre que elemento aplicar un estilo u otro

      CSS

      #app {
        background-color: green;
      }
      .primera-clase {
        background-color: yellowgreen;
      }
      div[segunda="segunda"] {
        border-style: groove;
        border-width: 10px;
      }
      .container > div {
        border-radius: 50%;
        background-color: lightblue;
        height: 100px;
        width: 100px;
      }
      .primera-clase ~ .segunda-clase {
        align-self: flex-start;
      }
      #app + div {
        align-self: flex-end;
      }
      
      HTML

      div id="app"div
      div class="primera-clase"div
      div class="segunda-clase" segunda="segunda"div
      div class="primera-clase"div
      div class="segunda-clase"div
