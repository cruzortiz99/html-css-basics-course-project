Flexbox

Flexbox inicio como soluci√≥n al problema que se presentaban con los layouts, ya que era muy complejo garantizar el posicionamiento de los elementos, antes de flexbox y posteriormente CSS Grid.

Flexbox permite alinear los elementos en 1 dimension, es decir vertical u horizontal, para crear layouts en 2 dimensiones se hacen arreglos de div y diferentes direcciones del flexbox

      .container {
        display: flex;
        flex-direction: column;
        justify-content: space-between;
        align-items: center;
      }
      .container div {
        background-color: lightblue;
        height: 100px;
        width: 100px;
      }
      .container div:nth-child(4n + 2) {
        align-self: flex-start;
      }
      .container div:nth-child(4n + 4) {
        align-self: flex-end;
      }
